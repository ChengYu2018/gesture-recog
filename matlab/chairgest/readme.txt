How to run
==========

1) Start MATLAB and change the directory to the main source folder: chairgest-yingyin.
The program has been tested with MATLAB R2013a.

2) In the Command Window type:

chairgestmain(raw_data_base_folder)

where `raw_data_base_folder` is the the folder containing RAW data for all the
users.

3) The program takes approximately 3.5 hours to finish and will output a txt
file "hmm_Converted_1.txt" in the main source folder.

4) For evaluation, the Converted data folder should be used. The output file
indicates that the Datatype is Converted and the MainSensor is Xsens.
